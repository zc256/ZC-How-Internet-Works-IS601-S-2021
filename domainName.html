Domain Names


Definition

  • A unique set of characters that identifies a specific website (similar to a house having a street address
  • When a user types in a domain name into a web browser, the browser communicates with the Domain Name Server (DNS) in order to find the corresponding website. This in turns allows the DNS to retrieve the website and be able to display it to the user

How to Read Domain Names

  • Each domain name has a Top Level Domain (TLD), such as .com or .net, and also contains a subdomain
  • Example: for example.com, the .com bit is the TLD, and the “example” bit is the subdomain. Altogether, this is a valid domain name that a user can use to visit the website.
  • Domain names can also include additional subdomains, such as en.example.com. This is a subdomain of example.com, and allows the user to view the English version

How do they work?

  • Domain names allow users to remember an easy set of words in order to access a website
  • Every website on the internet has a corresponding IP address that consists of a long string of numbers (IPv4) or a combination of letters and numbers (IPv6)
  • Instead of typing these hard to remember strings, the web browser connects to the DNS whenever a domain name is typed in. The DNS would then perform a lookup to find the corresponding IP address of the website the user wants to visit

Examples

Below are examples of domain names

  • en.wikipedia.org
    • ".org"-TLD, first-level domain name
    • "wikipedia"-Domain name (customizable), second-level domain name
    • "en"-English version (customizable), third-level domain name
  • njit.edu
    • ".edu"-TLD, first-level domain name
    • "njit"-Domain name (customizable), second-level domain name

Below are some examples of top-level domains (TLD)

  • ".com"-Commercial
  • ".org"-Organization
  • ".net"-Network
  • ".edu"-Education
  • ".gov"-Government

You can find more information at Wikipedia-Domain name and Wikipedia-List of Internet top-level domains


Why do we use Domain Names?

  • Makes it easy for users to find the websites they want to visit
  • The IP address of each website is not easy to remember
