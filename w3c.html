W3C


Definition

  • International community that works together to develop web standards
  • Its mission is to lead the World Wide Web to its full potential through the development of protocols and guidelines to ensure the long-term growth of the Web
  • Design principles
    • Web for All: W3C wants to make the sharing of knowledge available to all people regardless of hardware, software, network infrastructure, etc.
    • Web on Everything: With the growth of devices that can access the Web, this means that the Web should be able to handle this growth
  • Vision
    • Web for Rich Interaction: W3C aims to continue its support of the Webâ€™s evolution through building a strong architecture and design principles
    • Web of Data and Services: There are differing views in terms of the Web, as some see it as a massive collection of linked data while others view it as a giant collection of services that exchange communications W3C sees these views as complementary, and depending on the application one of these views can fit
    • Web of Trust: W3C realizes that people communicate through the Web, and trusts that their information is secure With this, it is vital that technology design fosters trust and confidence, and that there is support to handle complex interactions globally across many users

Examples

Some examples of W3C standards can be found at W3C Standards. Some examples include:

  • Web Design and Applications
    • HTML: with the "a" tag within a paragraph, the text within the "a" tags will be rendered on screen. When the text is clicked on, the user will be brought to the site within the "a href=" tag
  • Web of Devices
    • Graphics and Layout: features within the application need to be compelling and smooth as an interface
    • Data Storage: tech needs to be able to save properly, export content, and integrate the data from the multitude of files, etc.
