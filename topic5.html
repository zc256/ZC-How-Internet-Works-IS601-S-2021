W3C, HTML, and CSS

In order for web pages to be easy to use and readable, there needs to be a structure in place that allows for multiple elements to fit altogether nicely. W3C is the organization that sets the standards for what is accecptable with HTML and CSS. Without the standards, web pages would have different HTML/CSS formatting, and it would be hard for developers to follow. In terms of HTML and CSS, they provide the web page a structure and styling to allow the page to be user-friendly. With the use of tags, the developers can use texts, header, links, and other elements to convey whatever content they want to create. With CSS, it allows developers to add style to their pages, giving more creativity and flow to the overall page.
