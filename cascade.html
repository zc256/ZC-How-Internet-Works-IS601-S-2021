Cascading Style Sheet (CSS)


Definition

  • Stylesheet language used to describe how a given document written in HTML or XML looks like
  • Describe how elements should be rendered on the screen
  • Is a core language on the open web, and is standardized according to W3C specifications

Examples

The stylesheet also changes the styling of links. In this case, the links are bolded with blue coloring. And, the link color changes when it is either active or has been visited.


Why do we use CSS?

  • Defines the structure of a website, and allows for websites to have a different and unique look
  • Makes the website more user-friendly
  • Without CSS, websites would all look the same, and it would not be user-friendly or easy to use/read
